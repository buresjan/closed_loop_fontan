# Raw Data

Place the original `Aramburu_et_al_2024_Heliyon_e30404.zip` archive here when regenerating processed calibration/reference data.

This directory is ignored by Git because the archive and extracted MATLAB/Nektar binaries are third-party raw data and include files near or above GitHub's normal file-size limits. The tracked data products are generated under `data/processed/aramburu_2024/`.
